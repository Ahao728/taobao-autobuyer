# 淘宝秒杀助手 Pro (Taobao Sniper Pro)

这是一个基于 **Python** 和 **Selenium** 开发的桌面端淘宝抢购工具。它具备图形用户界面（GUI），支持自动登录、定时监控、自动勾选及高频提交订单功能。

## ✨ 功能特性

* **图形化界面**：基于 Tkinter 设计，支持运行日志实时显示与操作引导。
* **智能驱动管理**：支持手动指定 `msedgedriver.exe` 路径，内置驱动下载教程链接。
* **Cookie 持久化**：首次登录成功后自动保存 Cookie 到本地 `taobao_cookies.pkl`，后续运行可尝试免密登录。
* **精准抢购配置**：支持自定义商品关键字、精确到秒的抢购时间、以及点击频率控制。
* **繁忙退避逻辑**：程序能识别系统“繁忙/拥挤”提示，并自动执行冷却等待。

🚀 使用说明
1. 准备驱动
查看 Edge 浏览器版本（设置 -> 关于 Microsoft Edge）。

从 官方驱动页面 下载对应的 msedgedriver.exe。

将驱动文件放在本程序同级目录下或通过界面手动指定。

2. 启动步骤
运行程序：执行 python taobao1.py。

启动浏览器：点击 「1. 启动浏览器 & 登录」。在弹出的 Edge 窗口中完成淘宝扫码登录。

配置任务：

商品关键字：输入你想抢购的商品关键词。

抢购时间：设置目标时间（格式：22:00:00）。

高级配置：根据网络情况调整“点击间隔”，推荐 0.5 秒以上。

开始抢购：点击 「2. 开始抢购任务」，程序将自动进入购物车并在倒计时结束后执行任务。

📦 打包为 EXE (Windows)
如果您想将此工具打包成独立的 .exe 文件，请使用 PyInstaller：

Bash

pip install pyinstaller
pyinstaller --noconsole --onefile --name "淘宝秒杀助手" taobao1.py
注意：打包后的 .exe 位于 dist 文件夹内，运行时仍需将 msedgedriver.exe 放在同一目录下。
